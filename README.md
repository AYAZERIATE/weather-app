Weather Application
Overview

The Weather Application is a web-based project developed using HTML, CSS, and JavaScript.
It provides real-time weather information for selected cities and displays dynamic notifications based on weather conditions, such as rain alerts.

The application focuses on simplicity, usability, and clean user interface design while integrating real-world data through a public weather API.
Features

Real-time weather data retrieval by city name

Conditional weather notifications (e.g., rain alerts)

Animated notification panel for improved user experience

Sidebar navigation interface

Responsive and modern UI design
